# K-Expo LeadGen Lite

전시회 참가기업 리드 데이터를 활용한 Lead Scoring 및 Streamlit 대시보드 실습용 미니 프로젝트입니다.

## 프로젝트 구조

- **data/**: 전시회, 참가기업, 리드 점수 등의 샘플 데이터 파일 (.csv) 보관
- **scoring/**: 리드 스코어링 규칙 정의 및 평가 알고리즘 소스 코드 보관
- **dashboard/**: Streamlit을 활용한 데이터 시각화 및 인터랙티브 대시보드 웹 애플리케이션 코드 보관
- **docs/**: 프로젝트 관련 문서 및 설명 자료 보관
